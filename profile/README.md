### 주요 기능
1. 보호자의 허밍에서 주요 멜로디(Pitch) 추출
2. 생성형 AI MusicGen에 추출된 Pitch 데이터와 장르, 악기 프롬프트를 사용한 자장가 작곡
3. ChatGPT 모델을 사용한 사용자 맞춤형 자장가 가사 작사
4. 자장가별 아이 반응 기록
